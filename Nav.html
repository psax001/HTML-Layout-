Course Overview

Introduction to Web Technologies

This course provides a comprehensive introduction to web technologies, covering both front-end and back-end development. Students will learn HTML, CSS, JavaScript, and various frameworks essential for modern web development.

Through practical assignments and projects, students will gain hands-on experience in creating responsive, interactive, and accessible web applications.

Learning Objectives

By the end of this course, students will be able to:

  • Design and implement semantic HTML documents
  • Style web pages using CSS and responsive design principles
  • Create interactive user experiences with JavaScript
  • Understand client-server architecture and HTTP
  • Implement basic server-side functionality
  • Deploy web applications to hosting platforms

Course Syllabus

Week 1-2: HTML Fundamentals

Introduction to HTML5, document structure, semantic elements, forms, and accessibility best practices.

Week 3-4: CSS and Responsive Design

CSS selectors, box model, layouts, flexbox, grid, responsive design, and media queries.

Week 5-7: JavaScript Essentials

JavaScript syntax, DOM manipulation, events, asynchronous programming, and AJAX.

Week 8-10: Back-End Development

Introduction to server-side programming, databases, API development, and authentication.

Week 11-12: Modern Web Frameworks

Introduction to popular front-end and back-end frameworks and libraries.
